Software that Sucks

“How sad to see two leading Democrats fall for, even feed the lie that they can create user-oriented software for free.”

- Dave Winer, Scripting News, 12/24/2003

We have recently learned that the following software is not oriented to users. All users should stop using these products immediately.

  • Mail to the Future (spare time)
  • Friendster (ads)
  • NetNewsWire Lite (trial version)
  • iTunes (bundled)
  • SubEthaEdit (volunteers)
  • Creative Commons (grant)
  • OpenSecrets (donations)

Once again, the only way user-oriented software can be made is if users purchase it. Anything else is physically impossible.
